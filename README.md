# 게시판 만들기 [Summernote]
> 게시글 작성 과정서 화면에 나타나는 모양 그대로 출력물이 나오는 Summernote editor를 사용한 프로젝트 입니다.

## 구현 기능
---
* 게시글 작성 CRUD 기능, 조회수
* 게시글 검색: 키워드 검색[제목, 게시글]
* Swagger로 REST API 문서 관리
